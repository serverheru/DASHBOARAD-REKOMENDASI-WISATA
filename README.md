# DASHBOARAD-REKOMENDASI-WISATA
TOP DISTINASI WISATA PULAU JAWA
## Overview

Proyek ini merupakan dashboard Business Intelligence yang bertujuan untuk membantu pengguna menemukan rekomendasi tempat wisata di Kota Pulau Jawa dan sekitarnya. Dashboard menampilkan informasi wisata secara visual dan interaktif sehingga memudahkan pengguna memahami popularitas destinasi, kategori wisata, serta lokasi penyebaran tempat wisata.

Dashboard ini dikembangkan sebagai proyek portofolio Business Intelligence untuk menunjukkan kemampuan dalam proses analisis data, transformasi data, serta pembuatan visualisasi interaktif.

## Tujuan Proyek

* Menyediakan informasi tempat wisata di Pulau Jawa dalam bentuk visualisasi yang mudah dipahami.
* Membantu pengguna menemukan destinasi wisata berdasarkan kategori dan popularitas.
* Mengidentifikasi pola distribusi dan tren tempat wisata di Pulau Jawa.

## Dataset

Dataset yang digunakan berisi informasi terkait destinasi wisata di Pulau Jawa dengan beberapa atribut utama:

* Nama tempat wisata
* Kategori wisata
* Lokasi / wilayah
* Rating pengunjung
* Jumlah ulasan
* Fasilitas wisata

## Proses Analisis

Tahapan yang dilakukan dalam proyek ini meliputi:

1. **Data Cleaning**
   Menghapus data duplikat, menangani data kosong, serta memastikan konsistensi format data.

2. **Data Transformation**
   Mengubah struktur data agar sesuai untuk kebutuhan analisis dan visualisasi dashboard.

3. **Exploratory Data Analysis (EDA)**
   Mengeksplorasi pola data seperti distribusi kategori wisata, tingkat popularitas destinasi, dan persebaran lokasi.

4. **Data Visualization**
   Membangun dashboard interaktif untuk menyajikan insight secara visual.

## Fitur Dashboard

Dashboard menyediakan beberapa komponen visual utama:

* Distribusi jumlah wisata berdasarkan kategori
* Top destinasi wisata berdasarkan rating
* Analisis jumlah ulasan pengunjung
* Persebaran lokasi tempat wisata
* Filter interaktif berdasarkan kategori dan wilayah

## Insight yang Dihasilkan

Beberapa insight yang dapat diperoleh dari dashboard ini antara lain:

* Kategori wisata yang paling populer di Pulau Jawa
* Destinasi wisata dengan rating tertinggi
* Distribusi tempat wisata berdasarkan wilayah
* Rekomendasi wisata berdasarkan popularitas dan penilaian pengunjung

## Tools yang Digunakan
